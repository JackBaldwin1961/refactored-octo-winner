 🚀 Overview

 This project is a simple interactive image gallery built using HTML, CSS, and JavaScript.
 Users can add images dynamically by entering a URL, and remove them with a button.

 It demonstrates key front-end development concepts such as DOM manipulation, event handling, and dynamic content creation.

 🧠 Features

 • Add images using a URL input field

 • Dynamically display images in a gallery

 • Remove images with a "Remove" button

 • Responsive layout using Flexbox

 • Clean and simple UI styling

 🛠️ Technologies Used

 • HTML5 - Structure of the page

 • CSS3 - Styling and layout (Flexbox)

 • JavaScript (Vanilla) - Functionality and interactivity

 ⚙️ How It Works
 
 1. User enters an image URL into the input field

 2. Clicks the "Add Image" button

 3. JavaScript:

    • Gets the input value

    • Creates new DOM elements (div, img, button)

    • Appends them to the gallery

4. Each images includes a Remove button that deletes it from the gallery

💡 Key Concepts Demonstrated

• document.getElementById()

• createElement()

• appendChild()

• classList.add()

• addEventListener()

• onclick

• removeChild()

▶️ How to Run

1. Clone or download the repository

2. Open the folder in VS Code

3. Run using Live Server

4. Enter an image URL and click Add Image
