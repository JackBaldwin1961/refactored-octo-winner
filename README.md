# Front-End Web Development Projects (Jack Baldwin)

This repository contains a collection of front-end projects i've built while learning HTML, CSS and JavaScript. Each project focuses on different core skills such as responsive layouts, UI design, forms, and writing with embedded maps.

## Projects

### 1) Personal Portfolio Website
A responsive portfolio site to showcase projects, skills, and contact details.
**Tech:** HTML, CSS (Flexbox/Grid))

### 2) Cyberpunk Gaming Dashboard UI
A cyberpunk-themed dashboard layout using cards, buttons and sections.
**Tech:** HTML, CSS (Ui layout)

### 3) Google Maps Embed / Styling
A small project showing how to embed a Google Map and style the container layout.
**Tech:** HTML, CSS, JavaScript

### 4) Contact Us Page
A simple contact page demonstrating form structure and layout.
**Tech:** HTML, CSS

### HTML Email Template
An email-friendly template using table layouts for compatability.
**Tech:** HTML (email), inline/table layout

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Git & GitHub
